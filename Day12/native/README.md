# Native Project - Shape Library

## Prerequisites

- CMake 3.10+
- C++17 Compiler (g++)

## Build Steps

mkdir build
cd build

cmake ..

cmake --build .

## Run

./shape_test

Windows:

shape_test.exe

## How to Run Tests

The executable itself acts as the test program.

Run:

shape_test.exe

Expected output:
- Shapes sorted by area
- First shape with area > 100
